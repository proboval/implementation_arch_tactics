# Paper Summaries

36 per-paper summary files covering key findings, methodology, and relevance to the thesis.

## Naming Convention

`[authorsurname][year]_[keyword].md` (e.g., `garlan1993_introduction.md`)

## Summary Structure

Each summary includes:
1. Problem and motivation
2. Gap addressed
3. Main contribution
4. Method and validation
5. Key findings
6. Challenges and limitations
7. Relevance to thesis
